# Selenium E-Commerce Automation Framework

## Project Overview

This project is an end-to-end automation framework developed for the SauceDemo web application using Selenium WebDriver, Java, TestNG, and Maven.

## Features Automated

* Login Functionality
* Add to Cart Functionality
* Checkout Process
* Logout Functionality
* Data-Driven Testing using TestNG DataProvider
* Excel Data Handling using Apache POI
* Screenshot Capture
* Extent Reports

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* Apache POI
* Extent Reports
* Git
* GitHub

## Framework Design Pattern

Page Object Model (POM)

1. Login with valid credentials
2. Add product to cart
3. Complete checkout process
4. Logout from application
